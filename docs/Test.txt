📄 File : sockets/reservationSocket.js
    วิธีเทสผ่าน Socket.io Client หรือ Frontend

    1️⃣ เปิด server ด้วย : node server.js หรือ npm start
    2️⃣ ใช้โปรแกรมเช่น wscat หรือ Socket.io Tester

    เชื่อมต่อ :
    > wscat -c ws://localhost:3000

    ส่ง event :
    > emit newReservation { "roomId": 2, "status": "BOOKED" }

    ผลลัพธ์ :
    Client ทุกคนที่ต่อ socket อยู่จะได้รับ event:
    > reservationUpdate { "roomId": 2, "status": "BOOKED" }

    ❌ ถ้าไม่เห็น log → ตรวจสอบว่า socket.io server เชื่อมต่อถูกหรือไม่
